Features

Everything you need to organize your research activities efficiently

Automatic File Organization

The application automatically creates month folders and saves daily logs with proper date-based naming (DD-MM-YYYY.txt).

Calendar Navigation

Easily navigate through your logs using the interactive calendar. Days with logs are highlighted for quick reference.

Dark/Light Mode

Toggle between dark and light themes based on your preference or working conditions.

Auto-Save

Your work is automatically saved every 30 seconds, preventing data loss.

Basic Formatting

Add bold text, italics, and bullet points to organize your log entries.

Easy Access

Quickly open your log folder directly from the application with one click.

Installation Guide

Get up and running quickly with these instructions

Option 1: Using the Executable (Recommended)
  1. Download the Windows executable from the link above
  2. Double-click the downloaded file to run the application
  3. No installation required - it's a portable application
  4. Your logs will be saved in your Documents folder
Option 2: Using Python
  1. Install Python 3.7 or newer from python.org
  2. Install dependencies: 
    pip install ttkbootstrap pillow
  3. Download the source code from GitHub
  4. Run the application: 
    python research_log_manager.py
