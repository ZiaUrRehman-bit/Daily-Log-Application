Research Log Manager

A professional desktop application for managing daily research logs

Overview

Research Log Manager is a Python-based desktop application designed for PhD scholars and researchers who need to maintain organized daily activity logs. It automatically organizes your logs by month and date, providing a clean interface for recording your research progress.

Download for Windows View Source Code

Features

Automatic File Organization

The application automatically creates month folders and saves daily logs with proper date-based naming (DD-MM-YYYY.txt).

Calendar Navigation

Easily navigate through your logs using the interactive calendar. Days with logs are highlighted for quick reference.

Dark/Light Mode

Toggle between dark and light themes based on your preference or working conditions.

Auto-Save

Your work is automatically saved every 30 seconds, preventing data loss.

Basic Formatting

Add bold text, italics, and bullet points to organize your log entries.

Installation

Option 1: Using Python

  1. Install Python 3.7 or newer
  2. Install dependencies: pip install ttkbootstrap pillow
  3. Run the application: python research_log_manager.py

Option 2: Standalone Executable

Download the pre-built executable for Windows from the link above. No Python installation required.
